# First-Batch Admission & Publishing Rules

## 首批 persona 准入规则

只有同时满足以下条件的 persona 才能进入首发：

1. 2025-2026 仍有明显公开关注度
2. 与其他首发 persona 风格差异足够大
3. 有足够公开资料支撑非空洞 skill
4. 关联席位与风格的市场归因虽不完美，但可被谨慎表达

## 最低证据覆盖要求

每个首发 persona 至少具备：

- 2 条 hard evidence 来源
- 1 条 interpretation 来源
- 2 个代表案例
- 1 段 uncertainty note
- 1 组 risk rail 与 responsibility copy

## 元数据最小集合

- persona id / name
- launch flag
- style tags
- associated seats
- confidence
- source counts
- case counts
- searchable tags

## Provenance 最小集合

每个关键结论至少记录：

- 来源 URL
- 公开时间
- 来源层级
- 是否直接可核验
- 是否有冲突信息

## Dual-Lens 输出规则

任何“风格优势”后面都必须给出对应的：

- 失败模式
- 误用方式
- 不适用场景
